# **App Name**: CuyoCrops

## Core Features:

- Price Ticker: Display real-time prices of fruits and vegetables from Mercado Cooperativo de Guaymall√©n using a classic stock ticker interface.
- Product Cards: Display prices and details of fruits and vegetables in card format.
- Admin Panel: Admin panel for uploading new produce and setting the prices
- Price Tracking: Track price changes over time and display the history visually using a charting library
- Database Storage: Persistent storage of uploaded produce and corresponding prices, linked with each entry's modification date and time.
- Data Validation Tool: Use AI to monitor user entered prices to ensure there are no data entry errors when updating the database.

## Style Guidelines:

- Primary color: Saturated emerald green (#50C878) evokes freshness and organic produce.
- Background color: Very light green (#F0FAF4) provides a clean, neutral backdrop.
- Accent color: Golden yellow (#FFD700) draws the eye and suggests value.
- Headline font: 'Playfair', a serif font for high-end feel
- Body font: 'PT Sans', a sans-serif font to pair with Playfair for body.
- Simple, flat icons representing each fruit and vegetable category.
- Responsive layout that adapts to different screen sizes, maintaining readability and usability on all devices.
- Smooth transitions and subtle animations to indicate price changes or updates.
